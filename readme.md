Projeto desenvolvido em Python com o objetivo de simular um sistema de cadastro e autenticação de usuários, permitindo registrar novos usuários e realizar login com validação de credenciais.
Este projeto foi criado como parte dos meus estudos em programação e desenvolvimento de software, aplicando conceitos importantes da linguagem Python e organização de código.


Funcionalidades:
Cadastro de novos usuários;
Sistema de login com validação de senha;
Armazenamento de usuários em arquivo JSON;
Verificação de usuários já existente;
Interface de execução via terminal;


Tecnologias Utilizadas:
Python 3;
JSON (armazenamento de dados);
Git;
GitHub;
VS Code;


Estrutura do Projeto
sistema-login-python/
│
├── main.py        # Interface principal do sistema
├── auth.py        # Lógica de autenticação
├── database.py    # Manipulação do banco de dados JSON
├── usuarios.json  # Armazenamento dos usuários
└── README.md


Como Executar o Projeto
1️⃣ Clone o repositório
git clone https://github.com/GabrielSouzass/sistema-login-python.git
2️⃣ Acesse a pasta do projeto
cd sistema-login-python
3️⃣ Execute o programa
python main.py
